Detailed Description

One addend of a coefficient of an expression in the frequency domain.

Field Documentation

struct frq_frqDomExpressionAddend_t* frq_frqDomExpressionAddend_t::pNext

An expression in the frequency domain is a linked list of these addends. The link to the next addend is made by this pointer. NULL for the last or a single addend.

Remarks:
It's essential to have this member as the very first one. Its address needs to be identical to the address of the entire struct.
