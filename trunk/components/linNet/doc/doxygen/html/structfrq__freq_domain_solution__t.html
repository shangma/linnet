Detailed Description

The solution object contains the solution of the LES in the frequency domain.

Field Documentation

unsigned int frq_freqDomainSolution_t::noReferencesToThis

A counter of references to this object. Used to control deletion of object.

const char* frq_freqDomainSolution_t::name

The name of this solution. The name corresponds with the name of the user defined result this solution object belongs to.

const tbv_tableOfVariables_t* frq_freqDomainSolution_t::pTableOfVars

The table of variables describes the knowns, unknowns and constants of the problem; among more, their names are found here.

signed int frq_freqDomainSolution_t::idxResult

The index of the user-demanded result, which this solution object represents. The index is related to the table of result definitions in the pci_circuit_t object inside of * pTableOfVars.
A value of -1 means that this object holds a solution for all the unknowns of the LES regardless of user requested sub-sets of variables.

frq_normalizedFrqDomExpression_t* frq_freqDomainSolution_t::pDenominator

The system determinant and the common denominator of all result terms.

frq_normalizedFrqDomExpressionMatrix_t frq_freqDomainSolution_t::numeratorAry

An array [noSolutions, noKnowns] of numerators of result terms. There's one such term for each known in each solution.

const char* ** frq_freqDomainSolution_t::numeratorNameAry

The names of all numerator expressions, used e.g. as variable names in the generated Octave script.

const char* ** frq_freqDomainSolution_t::denominatorNameAry

The names of all (final) denominator expressions, used e.g. as variable names in the generated Octave script.
