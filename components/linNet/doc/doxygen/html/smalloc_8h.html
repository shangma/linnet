static void* smalloc ( size_t  noBytes,
const char *const  fileName,
unsigned long  line 
)
inlinestatic

Safe malloc: The return value NULL of malloc can't be handled at all but safe abortion of this (nearly neglectible improbable) situation is a minimum requirement. This function does the same as malloc does but aborts the application in case of "no memory" with a (developer) hint where the problem occurs.

Returns
The pointer to the allocated memory is returned.
Parameters
noBytesThe number of requested bytes
fileNameThe name of the source file, where the function is used. Simply pass the macro FILE.
lineThe source code line number, where smalloc is called. Simply pass the macro LINE.
static char* stralloccpy ( const char *const  str)
inlinestatic

Copy a string into a newly allocated chunk of memory. Unavailability of memory is handled by error message and application abort, see void *smalloc(size_t, const char * const, unsigned long) for details.

Returns
The malloc allocate memory chunk, which is initialized with a copy of the passed string, is returned. Use void free(void*) to free the memeory after use.
Parameters
strThe string to copy.
