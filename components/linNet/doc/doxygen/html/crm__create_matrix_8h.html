Detailed Description

Generic implementation of a pair of functions that creates and deletes a rectangular array of elements with run-time determined size. The functions are implemented as macro #CRM_CREATE_MATRIX. Include this header and call the macro once in order to generate the code for the pair of functions.
The generated functions are declared by the other macro #CRM_DECLARE_CREATE_MATRIX. Call this macro to generate the prototypes in your code.
