Detailed Description

A user demanded result, an LTI system. Either a simple transfer function, which can be plotted as Bode diagram or a complete solution.

Field Documentation

const char* pci_resultDef_t::name

The name of the result. The name is used e.g. as part of the title of frequency response plots.

unsigned int pci_resultDef_t::noDependents

The number of dependent quantities in the result.

const char* pci_resultDef_t::dependentNameAry[PCI_MAX_SIZE_OF_RESULT_SETS]

The name(s) of the dependent quantities. Normally this is a set of unknowns. In the particular case of one dependent and one independent, this may also be a known if an inverse transfer function is requested.

const char* pci_resultDef_t::independentName

The name of the independent quantity. Normally this is a known. In the particular case of one dependent and one independent quantity, this may also be an unknown if an inverse transfer function is requested.
The value is NULL if a full result is demanded, which depends on all knows of the system.

pci_plotInfo_t* pci_resultDef_t::pPlotInfo

Some information supporting result plotting. This information is not directly used but forwarded to the plot backend. The information is optional; if not givven the pointer is NULL.
